Route 53

Route 53 key points:

  • KMS creates and manages encryption keys for a number of services or for your applications

  • KMS is up to FIPS 140-2 Level 2 compliant

  • KMS has the ability to automatically rotate out your keys with no need to re-encrypt

  • CloudTrail accesses audit keys' history

  • Hardware Security Module (HSM) is a specialized hardware for storing keys and is tamper-proof

  • Customer master keys (CMKs) are the primary resources in KMS

  • KMS stores master keys

  • Master Keys are used to encrypt data keys. This is called Envelope Encryption.

  • Symmetric keys are single keys using 256 bit encryption

  • Asymmetric uses two keys (public and private)
